# Say It For Me

> You know what you mean. We help you say it.

A clean, no-backend web app that helps people write difficult messages — saying no, breaking up, confronting someone, apologizing, negotiating, and more.

## How it works

1. Pick your situation
2. Describe what happened
3. Choose a tone
4. Pick an AI provider and paste your API key
5. Get a ready-to-send message

## Supported AI providers

- Anthropic (Claude)
- OpenAI (GPT-4o)
- Google (Gemini)
- Mistral
- Cohere
- xAI (Grok)

## No backend

Users bring their own API key. The app runs 100% client-side — no server, no data stored, no cost on your end.

## Run locally

Just open `index.html` in your browser. No build step needed.

## Deploy

Works with any static host:
- GitHub Pages
- Vercel
- Netlify

## Stack

- Vanilla HTML, CSS, JavaScript
- Google Fonts (Instrument Serif + DM Sans)
- Tabler Icons
